# ContentStudio Templates

Standard templates for all content production within the VitalMatrix Content Studio.

## Available Templates

1. Content-Brief-Template.md — Use when commissioning any new content item
2. W08-Review-Request-Template.md — Attach to every item entering W08 review queue
3. Evidence-Tier-Assignment-Template.md — Required for any content containing clinical claims
4. Content-Naming-Convention.md — Mandatory naming rules for all files
5. Section-README-Template.md — Template for section-level README files
