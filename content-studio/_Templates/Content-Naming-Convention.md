# Content Naming Convention

All files in the ContentStudio must follow Rule F:
File name must begin with specific words describing exactly what is inside.

## Format
SpecificContentDescriptor[_Version]_YYYY-MM-DD.ext

## Never begin with
- Window names (W05, W08, ClaudeCode)
- Project names (VitalMatrix, HHW)
- Person names (DrFaisal)
- Dates alone
- Generic labels (Output, Document, Script, Report, Draft, Template, Summary, Brief)

## Examples
- GOOD: PractitionerOnboarding_WelcomePack_2026-05-20.pdf
- GOOD: CascadeAtlas_BoothHandout_v2_2026-05-20.pdf
- BAD: W05_Output_2026-05-20.pdf
- BAD: VitalMatrix_Document_Draft.pdf
